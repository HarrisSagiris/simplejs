SimpleJS (SJS) Documentation

Introduction

SimpleJS (SJS) is a simplified programming language that compiles to JavaScript. It provides an intuitive syntax while maintaining full JavaScript compatibility.

Installation

  1. Download SimpleJS-Setup.msi from the releases page
  2. Run the installer
  3. Verify installation:
sjs --version

Basic Syntax

SJS uses simplified commands that compile to standard JavaScript:

Basic Commands

// Printing
print "Hello World"

// Loops
repeat 5 times {
    print "Hello"
}

// User Input
name = input "Enter your name:"

Imports and Exports

// Importing
import express from "express"
import { useState } from "react"

// Exporting
export function myFunction() {
    // Function code
}

Compilation Process

The SJS compiler performs the following steps:

  1. Lexical analysis of SJS code
  2. Syntax transformation using regex patterns
  3. Special handling for loops, database operations, and HTTP requests
  4. JavaScript code generation

Runtime Environment

The SJS runtime environment provides:

  • Automatic dependency injection (axios, mongoose)
  • Global error handling
  • Isolated execution context
  • Built-in performance optimization

API Reference

Built-in Commands

SJS Command JavaScript Equivalent Description
print console.log Output to console
input prompt User input
repeat for Loop construct
connect mongoose.connect Database connection
get axios.get HTTP GET request
post axios.post HTTP POST request
Important Note: SJS files must have the .sjs extension and be run using the SJS compiler: 
sjs myfile.sjs

Development Tools

  • VSCode Extension: Syntax highlighting and IntelliSense for .sjs files
  • SJS CLI: Command-line tools for compilation and execution
  • Debug Tools: Built-in source map support for debugging
