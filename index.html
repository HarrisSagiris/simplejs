Introduction

SimpleJS (SJS) is a high-level programming language that compiles to JavaScript. It provides simplified syntax while maintaining full JavaScript compatibility and functionality.

Language Syntax

Basic Syntax Transformations

// SimpleJS Syntax → JavaScript Equivalent
print "Hello"     → console.log("Hello")
input "Name?"     → prompt("Name?")
repeat 5 times    → for(let i = 0; i < 5; i++)

Database Operations

// MongoDB Connection
connect to mongodb://localhost/mydb
→ mongoose.connect("mongodb://localhost/mydb", {
    useNewUrlParser: true,
    useUnifiedTopology: true
})

// Database Operations
save user         → Model.save(user)
find users        → Model.find(users)

HTTP Requests

// GET Request
get from https://api.example.com
→ axios.get("https://api.example.com", {
    validateStatus: status => status < 500
})

// POST Request
post to https://api.example.com with data
→ axios.post("https://api.example.com", data, {
    validateStatus: status => status < 500
})

Core Features

Error Handling

SJS includes built-in error handling and promise rejection catching:

try {
    // Your SJS code
} catch (error) {
    print "Error occurred: " + error.message
}

Module System

SJS supports JavaScript's module system with simplified syntax:

// Importing
import express from "express"
import { useState } from "react"

// Exporting
export function myFunction() {
    // Function code
}

Compilation Process

The SJS compiler performs the following steps:

  1. Lexical analysis of SJS code
  2. Syntax transformation using regex patterns
  3. Special handling for loops, database operations, and HTTP requests
  4. JavaScript code generation

Runtime Environment

The SJS runtime environment provides:

  • Automatic dependency injection (axios, mongoose)
  • Global error handling
  • Isolated execution context
  • Built-in performance optimization

API Reference

Built-in Commands

SJS Command JavaScript Equivalent Description
print console.log Output to console
input prompt User input
repeat for Loop construct
connect mongoose.connect Database connection
get axios.get HTTP GET request
post axios.post HTTP POST request
Important Note: SJS files must have the .sjs extension and be run using the SJS compiler: 
sjs myfile.sjs

Development Tools

  • VSCode Extension: Syntax highlighting and IntelliSense for .sjs files
  • SJS CLI: Command-line tools for compilation and execution
  • Debug Tools: Built-in source map support for debugging
